- 组件引入外部样式方式
  1. 将外部样式写在组件内部
  2. externalClasses
     不能对外部样式进行修改
  3. styleIsolation: 'apply-shared'
     可以直接对外部样式进行修改
- 组件通信
  triggerEvent
    自定义组件触发事件
- 获取文件扩展名正则
  /\.\w+$/.exec(item)[0]
- rest参数
  rest参数（形式：...变量名），用于获取函数的多余参数，rest参数搭配的变量是一个数组，该变量将多余的参数放入数组中。
- 扩展运算符(spread)
  形式：...
  作用：将一个数组转为用逗号分隔的参数序列
